# Odin Recipes

This project is part of **The Odin Project Foundations** curriculum.

The goal of this project is to build a simple recipe website using only HTML. The website consists of a main index page that links to several recipe pages. Styling is intentionally kept to a minimum because the focus is on learning the fundamentals of HTML before moving on to CSS.

## Skills Demonstrated

By completing this project, I practiced:

* Writing HTML boilerplate from scratch
* Creating multiple HTML pages
* Structuring content with headings, paragraphs, lists, and images
* Creating links between pages using relative paths
* Organizing files and folders within a web project
* Using Git and GitHub to track project progress through meaningful commits
